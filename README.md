# Installing the project on local
```
    git clone https://github.com/dwedaz/bebas-akuntasi
```

```
    composer update -vvv
```

```
    ./vendor/bin/sail up
```

```
    ./vendor/bin/sail artisan migrate
```

```
    ./vendor/bin/sail artisan db:seed
```
